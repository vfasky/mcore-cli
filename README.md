# mcore-cli
A command line tool to generate mcore3 project.

## install
```bash
npm install -g mcore-cli
```

## usage

```bash
# view help
mcore-cli -h

# command
mcore-cli -i [project_path]  -n [project_name] -t [project_type] -p [port]
```

1. project_type: 'es6', 'coffee'
2. port:  The http port to run project
